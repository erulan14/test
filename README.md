1. Задача
Сделать компонент где можно выбрать интервал (от и до с датой и времени) интервал должен быть разделен

пример на фото 

![изображение](https://github.com/erulan14/test_nurdaulet/assets/42293127/5245d591-7a8e-4348-8a4e-c84009f3ea22)


поосле выбора интервала нужно сделать кнопку где будет выполнен POST запрос на api для получение отчета о поездки, 
ссылка на api: https://glot.kz/api/v1/report/{id}/

id для теста можете написать 5257

{
    "id": 5257,

    "format": "json",

    "from": 1761937200,
